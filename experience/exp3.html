Project Walkthrough

1. Data Sources & Context

The project started by collecting social listening data from platforms like Instagram and community-driven content such as 7KM Run Club. These sources provided real user reactions, comments, and engagement signals around Samsung campaigns and Galaxy Watch-related content.

2. Understanding Customer Perception

I examined how real users perceive Galaxy Watch through platforms like Reddit. Common sentiments included:

  • "An additional gadget for your smartphone" — seen as an ecosystem accessory, not a standalone product
  • "Luxury cool item, almost totally useless" — appealing design but limited practical value
  • Price sensitivity — users wait for deals rather than paying full price

These raw perceptions helped define what content themes resonate — and what falls flat.

3. Feature Engineering

I transformed three raw data sources into structured, model-ready features:

  • Campaign performance data (likes, comments, CTR, VTR) → Content Relevance Scores with activity-specific signals (Running, Sleep, Fitness)
  • Creative and content tags (fitness, sleep, design, promotion) → Content Attribute Scores (Health Function, Design, Promotion)
  • Social listening text (consumer context and perception) → Engagement Features (Impressions, Likes, CTR, VTR)

4. Modeling & Key Findings

I trained a Random Forest model targeting Engagement Rate and Click-Through Rate, using the engineered content relevance, attribute, and engagement features as inputs.

  • Content with higher Running Signal Scores showed a meaningful uplift in engagement
  • The running signal consistently appeared as a top feature in model importance analysis

5. Scenario Comparison (Uplift Simulation)

To validate the finding, I designed an A/B scenario simulation:

  • Scenario A — Low Running Signal Score with baseline content setup
  • Scenario B — High Running Signal Score with enhanced running-related elements

The result: strengthening running-related content signals led to a measurable uplift in predicted engagement compared to the baseline.

6. From Insight to Execution

The data-driven insight translated directly into Samsung's real content strategy:

  • Amplified 7KM Run Club community content on Instagram
  • Launched a Son Heung-min × Galaxy Watch 8 collaboration campaign ("A Day of Sonny")
  • Promoted the Running Coach feature as a core product message

These were the exact content themes our model identified as high-engagement drivers.

  • Working at the intersection of data and marketing — This project taught me how product analytics, user behavior, and content strategy all connect — and how ML can be the bridge between them.
